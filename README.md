Stack Implementation in C/C++

A simple implementation of Stack Data Structure using Array in C/C++.
This project demonstrates basic stack operations and menu-driven execution for better understanding.

* Features

Push an element to stack

Pop an element from stack

Peek/Top element

Display stack contents

Overflow & Underflow handling

Easy menu-driven interface

* Project Files
* Stack-Program
├── stack.c / stack.cpp   # Main source file
└── README.md

* Requirements

GCC / G++ compiler

Works on Windows / Linux / Mac

▶ How to Compile & Run
For C Program
gcc stack.c -o stack
./stack

For C++ Program
g++ stack.cpp -o stack
./stack
